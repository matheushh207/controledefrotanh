CONTROLE DE FROTA - PROJETO SIMPLES

Projeto 100% JavaScript (HTML + JS)
Sem servidor
Sem Node.js
Sem credenciais Google

COMO USAR:
1) Abrir a pasta no VS Code
2) Abrir o arquivo index.html
3) Rodar no navegador (duplo clique ou Live Server)
4) Preencher o formulário e salvar

Os dados são enviados diretamente para a planilha Google via Apps Script.
